# ESP32_SPIFFS_MisakiFNT
Arduino IDE library for Misaki Font (SPIFFS) in ESP-WROOM-32(ESP32).  
Beta ver1.0

これは8x8ドットの美咲フォントのライブラリで、SPIFFS ファイルシステム読み取り用です。
